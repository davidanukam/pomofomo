# Pomo Fomo website

Marketing site for [Pomo Fomo](https://github.com/davidanukam/pomofomo), a calm Pomodoro timer for Windows.

## Develop

```bash
cd website
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy on Vercel

Create a Vercel project with **Root Directory** set to `website`. Framework preset: Next.js.
